## How to upload

Filename should be formatted like below
```
{id}-{[paper_alias]_[paper_year]}/{user_name}.ipynb
```

### Example

1. If you have a single file
```
1-AlexNet_2012/changwoolab.ipynb
```

2. If you have multiple files
```
1-AlexNet_2012/changwoolab/*.py
```
